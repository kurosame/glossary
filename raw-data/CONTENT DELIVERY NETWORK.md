## titles

Content Delivery Network
CDN

## description

Web コンテンツをインターネット経由で配信するために最適化されたネットワーク

### 種類

- Fastly  
  ユーザの位置情報を検知し、もっとも近いサーバから Web コンテンツを配信できる  
  従来の CDN サービスでは不可能だった動的コンテンツもキャッシングできる

- Cloudinary  
  画像や動画の配信に特化した CDN  
  ブラウザによって最適な形式で返す  
  （Chrome なら webp 形式、Safari なら jpeg 形式など）
