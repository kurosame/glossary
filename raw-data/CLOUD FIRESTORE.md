## titles

Cloud Firestore

## description

NoSQL データベース  
コレクションにまとめられたドキュメントにデータを保存する  
iOS, Android, Web のオフラインサポート  
クエリはインデックス付きで高速  
書き込み時のトランザクションは完了するまで自動的に保持  
スケーリングは自動で行われる

### Firebase Realtime Database との違い

- 性能は Cloud Firestore が上
- iOS, Android のみのオフラインサポート
- スケールしたい場合は、複数のインスタンスにデータをシャーディングする必要がある
