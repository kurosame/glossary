## titles

Svelte

## description

Rich Harris によって開発された JS フレームワーク  
読みはスヴェルト

React や Vue.js のようにブラウザ上で直接動くフレームワークとは違い、Svelte はコンパイラーとなる  
`.svelte`ファイルを HTML/JS/CSS に変換するだけである

特徴は以下

- 仮想 DOM を持たない
- ファイル内に記述した CSS はスコープ付きとなる
- Svelte に Store が内包されている
