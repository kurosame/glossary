### assets と static の違い

- assets  
  Sass など webpack の loader を使って処理したいファイルはこちらに置く
- static  
  ここに置かれたファイルは公開ディレクトリにそのまま置かれる  
  プロジェクトのルート URL からアクセス可能  
  webpack などで処理する必要のないファイルを置く
